Project Objectives:
1. Develop an intuitive web-based platform for tracking anime and TV series progress.
2. Enable tracking of watched and pending episodes.
3. Provide an interactive and community-driven experience through custom clubs.
4. Ensure a smooth and responsive user interface.
5. Implement AI-powered recommendations based on user preferences.
   
Core Features Developed:
1. Menu Categories: Users can organize shows under:
o All
o Watching
o Completed
o On Hold
o Dropped
o Plan to Watch
2. Progress Tracking: Users can log watched episodes and track remaining episodes.
3. Custom Clubs & Discussion Boards: Users can create/join groups to discuss shows.
4. Spoiler Protection: Users can choose to hide/reveal spoilers.
5. Polls & Ratings: Club members can vote on episodes, characters, and storylines.
6. User Reviews & Comments: Users can leave reviews on specific shows.
7. Episode & Season Reminders: Alerts for upcoming episodes or scheduled watch times.
8. Streaming Integration: Direct links to streaming platforms where shows are available.
9. Watchlist Sharing: Users can share their lists with friends and community members.
10. Dark & Light Mode: Custom themes for personalized user experience.
11. Tags & Filters: Users can filter by genre, language, release year, etc.
12. Analytics Dashboard: Graphs and insights on watch history and time spent.
13. Admin Panel: Web-based control panel for managing discussions and user interactions.
14. Web Optimization: Ensuring fast performance and cross-browser compatibility.
